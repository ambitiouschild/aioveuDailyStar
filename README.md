花裤衩
https://github.com/PanJiaChen/vue-element-admin

1. 解释“花裤衩vue-element-admin”是什么
   "花裤衩vue-element-admin" 是一个开源项目，旨在提供一个快速搭建企业级中后台产品原型的解决方案。它基于 Vue.js 和 Element UI，内置了最新的前端技术栈，如 i18n 国际化解决方案、动态路由、权限验证等，并提炼了典型的业务模型，提供了丰富的功能组件。
2. 列举“花裤衩vue-element-admin”的主要特性和功能
   ‌国际化多语言‌：支持多种语言切换。
   ‌动态路由‌：根据用户权限动态生成路由。
   ‌权限验证‌：包括页面权限、指令权限等。
   ‌多种动态换肤‌：支持用户自定义主题。
   ‌动态侧边栏‌：支持多级路由嵌套。
   ‌动态面包屑‌：提供页面导航路径。
   ‌快捷导航(标签页)‌：方便用户快速切换页面。
   Svg Sprite 图标‌：提供丰富的图标库。
   ‌本地/后端 mock 数据‌：方便开发者进行模拟测试。
   Screenfull全屏‌：支持全屏模式。
   ‌自适应收缩侧边栏‌：根据屏幕大小自动调整侧边栏。
   ‌编辑器‌：支持富文本、Markdown、JSON 等多格式编辑。
   Excel 操作‌：支持导出、导入 Excel 文件，以及前端可视化 Excel。
   ‌表格‌：提供动态表格、拖拽表格、内联编辑等功能。
   ‌错误页面‌：提供 401、404 等错误页面模板。
   ‌多种组件‌：如头像上传、返回顶部、拖拽 Dialog 等。
